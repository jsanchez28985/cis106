# LS
* **Definition**
  * The ls command is used to list the files inside of a directory.
* **Usage**
  * The ls command is to files and the directories. It lists the current files, and the subdirectories.
* **Formula** 
  * ls is the basic command that can be run alone. LS can also be ran as ls -l, ls -a.
* **Examples**
  * ls -l will show the permissions, size, and last modification time for each file. 
  * ls -a will show the hidden files. 

# PWD
* **Definition**
  * The PWD command is a function that displays the absolute path of the current working directory
* **Usage**
  * PWD is used for displaying the current working directory. 
* **Formula**
  * pwd [options]
* **Examples**
  * see the current physical working directory 
   * pwd + -P

# CD
* **Definition**
  * CD short for change directory is a command used in linux to go from one directory to another. 
* **Usage**
  * Used for changing the current working directory. When no directory is given, cd changes the current working directory to the home directory of the current user.
* **Formula** 
  * cd [directory_path]
* **Examples**
  * change to the Documents directory
   * cd Documents
  * Navigate to a specific directory 
   * cd /home/user/Pictures

# What is a variable?

    A variable is a named storage location that holds a value. Numbers, text strings or other data is a variable. 

# How do I use a variable?

    In shell scripting variables are used by doing the following, COLOR="Red". 

# What is an environment variable?

    Environment variables are used by the shell to track specific system information and user information. 

# What is a user defined variable?

    User variables are created by the user and exist only in the script and subshell that runs the script. User variables allow you temporarily store data and use it throughout the script - just like any other programming language. 

# What is the root directory? 

    The root directory is the first directory in a file system. Root directory contains all the files and subdirectories that makes your computer. 

# What does "Parent directory" mean? 

    A parent directory is the directory that comes before the current directory that you are inside of. An example can be /home/user1/Downloads, here the parent is home/user1

# What does "Current working directory" mean? 

    Current working directory simply means that the directory you are inside of at the moment is the current working directory. 

# What is an absolute path? 

    An absolute path is the location of a file starting at the root of the file system. An example would be /home/jayshp_envy/Pictures/Gojo.jpeg. 

# What is a relative path? 

    A relative path is the location of a file starting from the current working directory or a directory that is located inside the current working directory. an example would be Music/BadBunny.mp3

# What is the difference between "Your home directory" and "The home directory"? 

    Your home directory is the folder that is assigned to a specific user that will holder their docs, pictures and files. THe home directory is the main directory that has all other users homes.
